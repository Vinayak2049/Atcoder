#  Concatenation Square Number

##  Problem Statement
AtCoDeer the deer has found two positive integers `a` and `b`.  
Determine whether the concatenation of `a` and `b` (in this order) is a **perfect square number**.

---

##  Input Format
a b

- `1 ≤ a, b ≤ 100`  
- Both are integers.

---

##  Output Format
- Print `Yes` if the concatenation of `a` and `b` is a perfect square.  
- Print `No` otherwise.

---

##  Approach
1. Read `a` and `b` as strings.  
2. Concatenate them (`a+b`) and convert to integer.  
   - Example: `a = "1"`, `b = "21"` → `"121"` → integer `121`.  
3. Compute the integer square root of the concatenated number.  
4. If `(root * root == num)`, then it is a perfect square → print `"Yes"`.  
5. Otherwise → print `"No"`.

---

##  C++ Solution
```cpp
#include <iostream>
#include <string>
#include <cmath>
using namespace std;

int main() {
    string a, b;
    cin >> a >> b;

    int num = stoi(a + b);

    int root = (int) sqrt(num);
    if (root * root == num) {
        cout << "Yes" << endl;
    } else {
        cout << "No" << endl;
    }

    return 0;
}
